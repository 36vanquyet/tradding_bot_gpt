# Release Notes v5.0.0

Ngày phát hành: 2026-04-03

## Tổng quan

Bản V5 nâng bot từ một khung paper-trading cơ bản sang một nền tảng gần thực chiến hơn, tập trung vào đồng bộ trạng thái thật từ sàn, khả năng chịu lỗi API tốt hơn, cơ chế bảo vệ vị thế và khả năng quan sát qua dashboard web.

## Điểm mới

- Đồng bộ position thật và open orders từ sàn qua CCXT
- Thêm `pending_orders` vào bot state
- Hỗ trợ `limit order` cho entry
- Thêm `trailing stop` trong engine
- Retry các lời gọi API có tính tạm thời như network timeout hoặc exchange unavailable
- Thêm dashboard web nội bộ và API `/api/status`
- Nâng chatbot:
  thêm `/help`, đổi ngôn ngữ `vi/en`, xử lý an toàn khi callback bị lặp

## Cấu hình mới

- `TRAILING_STOP_PCT`
- `ENTRY_ORDER_TYPE`
- `LIMIT_PRICE_OFFSET_PCT`
- `API_RETRY_ATTEMPTS`
- `API_RETRY_DELAY_SECONDS`
- `DASHBOARD_ENABLED`
- `DASHBOARD_HOST`
- `DASHBOARD_PORT`

## Thay đổi kỹ thuật chính

- Mở rộng `ExchangeAdapter` để hỗ trợ:
  `create_limit_buy`, `create_limit_sell`, `fetch_open_orders`, `sync`
- `PaperExchange` mô phỏng limit order và open order fill
- `CCXTAdapter` thêm retry wrapper và sync position/open orders
- `TradingEngine` cập nhật protective flow để xử lý trailing stop
- State store lưu thêm `language` và `pending_orders`

## Tài liệu bổ sung

- README được cập nhật theo trạng thái code hiện tại
- Bổ sung tài liệu mô tả chatbot tại `docs/chatbot-features.md`

## Ghi chú vận hành

- Trailing stop hiện được quản lý bởi bot ở tầng ứng dụng, chưa phải native trailing order của sàn
- Với một số sàn spot, dữ liệu position thật có thể phải fallback từ số dư asset nếu sàn không hỗ trợ `fetch_positions`
- Cần test kỹ trên testnet trước khi dùng tiền thật
